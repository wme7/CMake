# How to use `HDF5` with CMake

## installation of library

* In MacOS:
    ```console
    brew install HDF5
    ```
* In Linux (Ubuntu)
    ```console
    apt-get install HDF5
    ```

## Usage the `HDF5` library with cmake

1. cd /to/CMake_example/
2. Copy and paste the following instructions in your terminal:
    ```console
    mkdir build
    cd build
    cmake ..
    make
    ./test.run
    ```
