# How to use `Eigen3` with CMake

## installation of library

* In MacOS:
    ```console
    brew install Eigen3
    ```
* In Linux (Ubuntu)
    ```console
    apt-get install Eigen3
    ```

## Usage the `Eigen3` library with cmake

1. cd /to/CMake_example/
2. Copy and paste the following instructions in your terminal:
    ```console
    mkdir build
    cd build
    cmake ..
    make
    ./test.run
    ```
